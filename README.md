# Jonathan Diaz Portfolio

A dependency-free static portfolio website. It can be hosted directly with GitHub Pages; no build command, framework, database, authentication, or environment variables are required.

## Publish with GitHub Pages

1. Upload all files in this folder to the root of your GitHub repository.
2. In the repository, open **Settings > Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select the `main` branch and `/ (root)`, then save.

GitHub will provide the public URL after the first deployment finishes.

## Files

- `index.html` - website content and structure
- `styles.css` - complete responsive design
- `jonathan-diaz.jpg` - profile image
- `Resume.pdf` - downloadable resume
- `favicon.svg` - browser icon

Edit contact links and portfolio text directly in `index.html`. Design colors and layout are in `styles.css`.
