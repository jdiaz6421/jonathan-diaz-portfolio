Engineering decision

Preventing a silent storage failure

A missing NFS mount can leave a normal-looking local directory behind. Containers may then start successfully and write persistent data to the VM disk instead of the NAS.

I turned that failure mode into a preflight gate that checks Docker, verifies the mount with findmnt, checks disk capacity, and fails deployment when dependencies are unavailable.
